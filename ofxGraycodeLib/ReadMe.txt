========================================================================
    STATIC LIBRARY : ofxGraycodeLib Project Overview
========================================================================

This is a Visual Studio project definition to enable quick and easy inclusion of ofxGraycode within your projects

Instruction for use:

1. Add this project to your Visual Studio solution
2. Add "../../../addons/ofxGraycode/src" to your 'emptyExample'-like project's include path
3. Open the properties dialogue for your oF project, go to the top of the list on the left to find References. Click 'Add reference...' and select ofxGraycodeLib project.

NB: step 2 (and 3?) will be removed later when i've setup Property Sheets for this project, which can automatically transfer settings to relevant projects.
